# Monkefit Master Knowledge Base

This directory is the structured single source of truth for Monkefit AI.

## Structure
- company/
- products/
- pricing/
- service/
- sales/
- faq/
- decision-engine/
- ai/

## Rules
- Use YAML for structured knowledge cards.
- Every card must include metadata, version, status, owner, review date, sources, and confidence.
- Product prices from the public Shop section of monkefit.com must not be used as the official selling price.
- Internal COGS and profit data must not be stored in this public repository.
